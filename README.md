# Notes

Notebooks collection

## How to

- install?
  `yarn i`
  `pip install jupyter`
- run? `yarn dev`
- test? `yarn test`
- build? `yarn build`
- deploy? `yarn deploy`

## Uses

- [React](https://github.com/facebook/react)
- [jupyter nbconvert](https://github.com/jupyter/nbconvert)
- [ipynb-loader](https://github.com/n6g7/ipynb-loader)
- [hack.css](https://github.com/egoist/hack)
